# 🌸 RootliFocus: Grow Your Productivity 🌱

**RootliFocus** is a cozy productivity companion designed to turn your work sessions into a digital class. Connect with your Twitch community or stay focused solo while a cute character supports your progress.

---

## ✨ Features

* **Aesthetic Timer:** A clean Pomodoro clock with fully customizable themes.
* **Twitch Connectivity:** Real-time task synchronization with Twitch chat commands.
* **Character Customization System:** Fully modular character with interchangeable skins, hair and outfits.
* **Integrated Task List:** A lightweight to-do manager with persistent storage.

 ⚙️ Click the character to open the Settings menu.

---

## 📦 Installation

Go to the [Releases](../../releases) page and pick your preferred download:

**Option A — Installer (recommended):** `RootlifyFocus-X.X.X.exe`
- Double-click to install.

**Option B — Portable:** `RootlifyFocus-vX.X.X-portable.zip`
- Extract the zip, keep `assets/` next to the JAR, then run: `java -jar RootlifyFocus.jar`
- Requires Java 17+.

> Windows only.

For release/build info see [.github/workflows/README.md](.github/workflows/README.md).
